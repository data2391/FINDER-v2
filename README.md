## 🛠️ GUIDE D'INSTALLATION (FAST-TRACK)

L'installation automatise la création de l'environnement virtuel et la configuration des navigateurs "Headless".

### 🪟 WINDOWS (La méthode "One-Click")

1. Téléchargez et extrayez le dossier FINDER.
2. **Double-cliquez simplement sur `install_finder.bat`.**

> **Note :** Si Windows affiche une alerte, cliquez sur "Informations complémentaires" -> "Exécuter quand même".
> C'est juste le système qui a peur de ce qu'il va devenir. 🏰⚡

### 🐧 LINUX (BlackArch / Kali / Ubuntu)

1. Ouvrez votre terminal dans le dossier.
2. Lancez l'armurier :

```bash
chmod +x install_finder.sh && ./install_finder.sh
```

## 🚀 Lancement & Accès

1. Lancez le script de démarrage (ou `python server.py`).
